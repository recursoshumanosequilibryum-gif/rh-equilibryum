🔧 Configuração Inicial

Esta configuração é feita uma única vez pelo administrador. Todos os usuários salvarão dados no mesmo Google Drive.

1
Cole o Client ID do Google Cloud abaixo:
2
Clique em "Conectar ao Google Drive" e faça login com a conta da Equilibryum (a conta que ficará como dona dos dados).
3
Após conectar, o sistema abrirá normalmente e gerará um Token de Acesso para compartilhar com outros usuários.

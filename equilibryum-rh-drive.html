🔧 Configuração do Google Drive

Para salvar todos os dados e documentos no seu Google Drive, você precisa configurar uma vez só. Siga os passos abaixo:

1
Acesse console.cloud.google.com e faça login com sua conta Google.
2
Crie um projeto novo → vá em APIs & Services → Library → ative Google Drive API e Google Sheets API.
3
Vá em APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID. Tipo: Web application. Em "Authorized JavaScript origins" adicione a URL onde o sistema está hospedado (ou http://localhost para testes). Copie o Client ID.
4
Cole o Client ID abaixo e clique em Conectar:

💡 Após conectar, todos os cadastros e documentos serão salvos automaticamente no seu Google Drive, acessível de qualquer dispositivo.
